# Changing Table (Go)

## Описание
Changing Table - приложение на Go для обработки и преобразования данных с последующей записью в реляционную базу данных.

Проект разработан в рамках практики для изучения языка Go, работы с базами данных и построения backend-приложений.

---

## Функциональность
- Обработка входных данных  
- Преобразование данных по заданной логике  
- Запись данных в PostgreSQL  
- Базовая обработка ошибок  

---

## Стек технологий
- Go (Golang)  
- PostgreSQL  
- SQL  
- Docker  
- Postman  

### Внешние библиотеки:
- pgx (работа с БД)  
- mux (роутинг HTTP-запросов)
